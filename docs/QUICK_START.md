## Where are my recordings?

All recordings are saved to:
```
C:\Users\<YourName>\Videos\FluxRecordings\
```

Click "üìÅ Open Folder" in the app to access them quickly!

## Building Standalone Executable

Want to share the app or run without Python?

1. Run `build.bat`
2. Wait for build to complete
3. Find `FluxRecorder.exe` in the `dist` folder
4. Share or run the .exe file - no Python needed!
